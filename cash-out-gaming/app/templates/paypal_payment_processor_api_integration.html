PayPal Payment Processor API Integration

Integrate our PayPal payment processor API into your application to process payments.

API Endpoints

Endpoint Method Description
/paypal-payment-processor/process POST Process a payment
/paypal-payment-processor/refund POST Refund a payment

API Request Examples

Here are some examples of API requests you can make to our PayPal payment processor API:

            // Process a payment
            curl -X POST \
              https://example.com/paypal-payment-processor/process \
              -H 'Content-Type: application/json' \
              -d '{"amount": 10.99, "currency": "USD"}'

            // Refund a payment
            curl -X POST \
              https://example.com/paypal-payment-processor/refund \
              -H 'Content-Type: application/json' \
              -d '{"amount": 10.99, "currency": "USD"}'

API Response Examples

Here are some examples of API responses you can expect from our PayPal payment processor API:

            // Process a payment response
            {
              "transaction_id": 1,
              "status": "paid"
            }

            // Refund a payment response
            {
              "transaction_id": 1,
              "status": "refunded"
            }
