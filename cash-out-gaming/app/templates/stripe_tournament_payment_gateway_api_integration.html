Stripe Tournament Payment Gateway API Integration

Integrate our Stripe tournament payment gateway API into your application to process payments for tournaments.

API Endpoints

Endpoint Method Description
/stripe-tournament-payment-gateway/charge POST Charge a customer's credit card for a tournament
/stripe-tournament-payment-gateway/refund POST Refund a customer's credit card for a tournament

API Request Examples

Here are some examples of API requests you can make to our Stripe tournament payment gateway API:

            // Charge a customer's credit card for a tournament
            curl -X POST \
              https://example.com/stripe-tournament-payment-gateway/charge \
              -H 'Content-Type: application/json' \
              -d '{"amount": 10.99, "currency": "USD", "tournament_id": 1}'

            // Refund a customer's credit card for a tournament
            curl -X POST \
              https://example.com/stripe-tournament-payment-gateway/refund \
              -H 'Content-Type: application/json' \
              -d '{"amount": 10.99, "currency": "USD", "tournament_id": 1}'

API Response Examples

Here are some examples of API responses you can expect from our Stripe tournament payment gateway API:

            // Charge a customer's credit card for a tournament response
            {
              "transaction_id": 1,
              "status": "paid"
            }

            // Refund a customer's credit card for a tournament response
            {
              "transaction_id": 1,
              "status": "refunded"
            }
