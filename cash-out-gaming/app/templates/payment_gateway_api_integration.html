Payment Gateway API Integration

Integrate our payment gateway API into your application to process payments.

API Endpoints

Endpoint Method Description
/payment-gateway/charge POST Charge a customer's credit card
/payment-gateway/refund POST Refund a customer's credit card

API Request Examples

Here are some examples of API requests you can make to our payment gateway API:

            // Charge a customer's credit card
            curl -X POST \
              https://example.com/payment-gateway/charge \
              -H 'Content-Type: application/json' \
              -d '{"amount": 10.99, "currency": "USD"}'

            // Refund a customer's credit card
            curl -X POST \
              https://example.com/payment-gateway/refund \
              -H 'Content-Type: application/json' \
              -d '{"amount": 10.99, "currency": "USD"}'

API Response Examples

Here are some examples of API responses you can expect from our payment gateway API:

            // Charge a customer's credit card response
            {
              "transaction_id": 1,
              "status": "paid"
            }

            // Refund a customer's credit card response
            {
              "transaction_id": 1,
              "status": "refunded"
            }
