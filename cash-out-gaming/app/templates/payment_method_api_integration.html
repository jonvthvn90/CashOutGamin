Payment Method API Integration

Integrate our payment method API into your application to retrieve payment methods.

API Endpoints

Endpoint Method Description
/payment-method GET Retrieve payment methods
/payment-method/{id} GET Retrieve a payment method by ID
/payment-method POST Create a new payment method
/payment-method/{id} PUT Update a payment method
/payment-method/{id} DELETE Delete a payment method

API Request Examples

Here are some examples of API requests you can make to our payment method API:

            // Retrieve payment methods
            curl -X GET \
              https://example.com/payment-method \
              -H 'Content-Type: application/json'

            // Retrieve a payment method by ID
            curl -X GET \
              https://example.com/payment-method/1 \
              -H 'Content-Type: application/json'

            // Create a new payment method
            curl -X POST \
              https://example.com/payment-method \
              -H 'Content-Type: application/json' \
              -d '{"name": "Credit Card", "description": "A credit card payment method"}'

            // Update a payment method
            curl -X PUT \
              https://example.com/payment-method/1 \
              -H 'Content-Type: application/json' \
              -d '{"name": "Updated Credit Card", "description": "An updated credit card payment method"}'

            // Delete a payment method
            curl -X DELETE \
              https://example.com/payment-method/1 \
              -H 'Content-Type: application/json'
