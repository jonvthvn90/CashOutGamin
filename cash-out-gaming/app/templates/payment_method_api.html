Payment Method API

Payment Method API

Use our payment method API to integrate our payment method into your application.

API Endpoints

Endpoint Method Description
/payment-method/create POST Create a new payment method
/payment-method/update POST Update an existing payment method
/payment-method/delete POST Delete a payment method

API Documentation

For more information on our payment method API, please refer to our API documentation.

API Request Examples

Here are some examples of API requests you can make to our payment method API:

            // Create a new payment method
            curl -X POST \
              https://example.com/payment-method/create \
              -H 'Content-Type: application/json' \
              -d '{"name": "Credit Card", "description": "A credit card payment method"}'

            // Update an existing payment method
            curl -X POST \
              https://example.com/payment-method/update \
              -H 'Content-Type: application/json' \
              -d '{"id": 1, "name": "Updated Credit Card", "description": "An updated credit card payment method"}'

            // Delete a payment method
            curl -X POST \
              https://example.com/payment-method/delete \
              -H 'Content-Type: application/json' \
              -d '{"id": 1}'

API Response Examples

            // Create a new payment method response
