Cash App Payment Method API Integration

Integrate our Cash App payment method API into your application to retrieve payment methods.

API Endpoints

Endpoint Method Description
/cash-app-payment-method GET Retrieve payment methods
/cash-app-payment-method/{id} GET Retrieve a payment method by ID

API Request Examples

Here are some examples of API requests you can make to our Cash App payment method API:

            // Retrieve payment methods
            curl -X GET \
              https://example.com/cash-app-payment-method \
              -H 'Content-Type: application/json'

            // Retrieve a payment method by ID
            curl -X GET \
              https://example.com/cash-app-payment-method/1 \
              -H 'Content-Type: application/json'

API Response Examples

Here are some examples of API responses you can expect from our Cash App payment method API:

            // Retrieve payment methods response
            {
              "payment_methods": [
                {
                  "id": 1,
                  "name": "Cash App"
                }
              ]
            }

            // Retrieve a payment method by ID response
            {
              "id": 1,
              "name": "Cash App"
            }
