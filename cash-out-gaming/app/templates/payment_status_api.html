Payment Status API

Use our payment status API to retrieve the status of a payment.

API Endpoints

Endpoint Method Description
/payment-status GET Retrieve the status of a payment

API Request Examples

Here are some examples of API requests you can make to our payment status API:

            // Retrieve the status of a payment
            curl -X GET \
              https://example.com/payment-status \
              -H 'Content-Type: application/json' \
              -d '{"payment_id": 1}'

API Response Examples

Here are some examples of API responses you can expect from our payment status API:

            // Retrieve the status of a payment response
            {
              "payment_id": 1,
              "status": "paid"
            }
