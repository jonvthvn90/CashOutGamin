Twitch Payment Status API Integration

Integrate our Twitch payment status API into your application to retrieve the status of a payment.

API Endpoints

Endpoint Method Description
/twitch-payment-status GET Retrieve the status of a payment
/twitch-payment-status/{id} GET Retrieve the status of a payment by ID

API Request Examples

Here are some examples of API requests you can make to our Twitch payment status API:

            // Retrieve the status of a payment
            curl -X GET \
              https://example.com/twitch-payment-status \
              -H 'Content-Type: application/json'

            // Retrieve the status of a payment by ID
            curl -X GET \
              https://example.com/twitch-payment-status/1 \
              -H 'Content-Type: application/json'

API Response Examples

Here are some examples of API responses you can expect from our Twitch payment status API:

            // Retrieve the status of a payment response
            {
              "payment_status": "paid"
            }

            // Retrieve the status of a payment by ID response
            {
              "id": 1,
              "payment_status": "paid"
            }
